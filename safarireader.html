I have successfully managed to trigger the reader with this code.
The critical steps are the following, taken from mathiasbynens.be/notes/safari-reader:

  1. A tag such as < section > or < div > must wrap around the entire code that is to appear in reader
  2. There must be at least 5 elements (< br > lines count as single elements) within the wrapper element
  3. The file must not be a local file
